Disjoint-set data structures for JavaScript.
See [docs](https://union-find.github.io/non-contiguous/index.html).
Parent is
[js-data-structures](https://github.com/make-github-pseudonymous-again/js-data-structures).

```js
find( a ) === find( b ) ; // false
union( A , B ) ;
find( a ) === find( b ) ; // true
```
